# 📖 Overview

**Bug Detection Agent** is an intelligent AI-powered code review platform that helps developers identify software bugs, security vulnerabilities, and code quality issues before deployment.

The application allows users to upload one or multiple source code files, select the desired bug detection categories, and leverage **Claude AI** to perform in-depth code analysis. The detected issues are presented with severity levels, detailed explanations, code snippets, suggested fixes, project health metrics, and downloadable reports.

This tool streamlines the code review process by automating software defect detection through Artificial Intelligence.
